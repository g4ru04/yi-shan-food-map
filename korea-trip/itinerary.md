# 韓國首爾 4 天 3 夜行程規劃

> 10/6 (一) 19:35 抵達仁川 → 10/10 (五) 12:35 離開仁川
> 10/8 (三) 19:45 高陽體育館 The Weeknd 演唱會
> 3 人 2 間房

---

## 住宿推薦

### 首選：忠武路站公寓式飯店

| 項目 | 內容 |
|------|------|
| 位置 | 地鐵忠武路站 5 號出口直達 |
| 價格 | 3人2間房（三人房+雙人房）四晚估 NT$20,000~27,000（實際依房型查詢） |
| 設施 | 有廚房、轉接頭、行李秤、掛燙機 |
| 行李 | 1 樓免費寄放 |

**為什麼選這間？**

| 優勢 | 說明 |
|------|------|
| 地鐵3號線 直達高陽 | 忠武路 → 三松站（汗蒸幕）約 50 分鐘，→ 高陽體育館（演唱會）約 55 分鐘，**全程不需轉車** |
| 地鐵4號線 到明洞一站 | 忠武路 → 明洞站只要一站（2 分鐘），或步行 5-8 分鐘 |
| 機場交通 | 忠武路 → 地鐵4號線 → 首爾站（4 站 8 分鐘）→ AREX 機場快線（43 分鐘） |
| 生活機能 | 附近有新堂辣雞爪街、24H 豬肉湯飯 |
| 性價比 | 所有推薦住宿中 CP 值最高 |

### 備選：明洞斯坦福飯店

- 乙支路入口站步行 2 分鐘
- 樓下就是明洞商圈，購物最方便
- 樂天百貨/免稅店對面
- 缺點：到高陽需轉車（地鐵2號線 → 地鐵3號線）

> **重點**：AQUAFIELD 和演唱會都在地鐵3號線（橘線）沿線，從忠武路出發全程不用轉車！
>
> **什麼是地鐵？** 首爾地鐵（지하철）就像台北捷運，是地下鐵路系統，用號碼和顏色區分路線。買交通卡感應進站即可搭乘。
>
> **末班車注意**：地鐵3號線末班車從高陽方向約 **23:20~23:40** 發車。演唱會約 22:00 結束，正常情況有 1 小時以上的緩衝。但若安可延長或散場人潮太多：
> - 備好 **Kakao T**（韓版 Uber），計程車回忠武路約 **₩25,000~35,000**（約 NT$600~850）
> - 3 個人搭計程車分攤下來其實也不貴

## 交通路線總覽

> 首爾地鐵（지하철）= 台北捷運，用號碼和顏色區分路線
> 營運時間：約 **05:30~23:40**，末班車後請用 Kakao T 叫車

```
忠武路站（地鐵3號線 橘線 / 地鐵4號線 藍線）
│
├── 地鐵4號線（藍線）→ 明洞站（1 站 2 分鐘）→ 購物
├── 地鐵4號線（藍線）→ 首爾站（4 站 8 分鐘）→ AREX 機場快線
├── 地鐵3號線（橘線）→ 三松站（約 50 分鐘）→ AQUAFIELD 汗蒸幕
├── 地鐵3號線（橘線）→ 高陽（約 55 分鐘）→ 演唱會
│
└── 轉乘
    ├── → 地鐵2號線（綠線）弘大入口（約 20 分鐘）→ 弘大購物
    └── → 地鐵2號線（綠線）聖水（約 35 分鐘）→ 聖水逛街
```
