📦 批次匯入

支援 CSV 或 JSON 檔。第一次用建議先下載範例照著填。

檔案格式說明

欄位（順序不限）：name, lat, lon, review, rating, google_rating, google_url, author, image_url, visited_at, category, is_restaurant, is_closed

name / lat / lon 必填，其餘可留空。

visited_at 造訪時間填如 2026-06-07 17:00。

is_restaurant 留空＝餐廳；非餐廳填 false 或 否。

is_closed 留空＝營業中；已歇業填 true 或 是。

批次匯入的 image_url 請直接填圖片網址（批次無法上傳檔案，檔案上傳請用左邊表單）。
